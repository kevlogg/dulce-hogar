# Dulce Hogar | Diseño & Estilo

Sitio web de una sola página (single-page) para la tienda argentina premium de decoración y muebles.

## 🏠 Sobre Nosotros

Dulce Hogar es una tienda especializada en muebles y decoración premium. Operamos desde Moreno, Buenos Aires, y enviamos a todo el país.

## ✨ Características

- Sitio completamente responsivo (mobile-first)
- Diseño elegante con paleta de colores cálidos
- Secciones: Inicio, Nosotros, Productos, Testimonios, Contacto
- Formulario de contacto funcional
- Integración con WhatsApp para consultas
- Animaciones suaves y efectos visuales
- Sin dependencias externas (solo Google Fonts)

## 🚀 Cómo Usar

1. Abre `index.html` directamente en tu navegador
2. No requiere servidor ni instalación de dependencias

## 📱 Responsividad

Optimizado para:
- Mobile (320px+)
- Tablet (768px+)
- Desktop (1024px+)
- Wide screens (1440px+)

## 💻 Tecnología

- HTML5 semántico
- CSS3 con variables CSS
- JavaScript vanilla
- SVG inline para iconografía
- Google Fonts (Cormorant Garamond, Playfair Display, Raleway)
